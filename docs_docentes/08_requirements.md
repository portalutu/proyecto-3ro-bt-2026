# 8. Documento de Requisitos

## Template

**Descripción general:**

**Requisitos funcionales:**

- RF1
- RF2

**Requisitos no funcionales:**

- RNF1
- RNF2

------------------------------------------------------------------------------------------------------------------------------

## Ejemplo

**Descripción:**
Sistema web para registrar préstamos.

**Requisitos funcionales:**

- RF1 Registrar libros
- RF2 Registrar préstamos

**Requisitos no funcionales:**

- RNF1 Sistema web
- RNF2 Soporte para múltiples usuarios
